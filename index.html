Personal Values Board

Keyboard flow: j/k select ↓/↑, c i n x u send card to Core / Important / Nice / Not / Unsorted (stay here, auto-advance), C I N X U jump focus to that column, h/l send card one column ←/→ (stay), H/L send card one column ←/→ and follow it, J/K reorder in lane, g/G top/bottom, r shuffle Unsorted, e preview export, Export .txt / .csv to download, Import .csv to reload.

Lowercase = send the card (classification mode). Uppercase = move your focus (navigation mode). Drag-and-drop with mouse still works.
